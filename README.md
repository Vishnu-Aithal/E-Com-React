> ### Tech Stack Used:

-   HTML5
-   CSS3
-   ReactJS
-   MockBee - Fake Backend

---

> ### Features:

-   All pages are responsive
-   Product Listing
-   Landing Page
-   Authentication
-   Routes
-   Sorting and Filtering
-   Cart Management
-   Wishlist Management
-   Cart Summary
-   Loaders

---

> ### How to run locally:

1. Fork the Repo
2. cd root directory
3. run npm install to install dependencies
4. run npm start to spin up a local dev server
